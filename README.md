# SecondBrain

AI-powered personal life operating system — health, career, wealth, and knowledge in one place.

## Tech Stack

- **Web**: Next.js 15 (App Router) + TypeScript + Tailwind CSS
- **Auth**: Clerk
- **Database**: PostgreSQL + Prisma ORM
- **Cache**: Redis
- **AI**: Claude claude-sonnet-4-6 (Anthropic)
- **Monorepo**: Turborepo + pnpm workspaces

## Prerequisites

- Node.js 20+ (`nvm use 20`)
- pnpm (`npm i -g pnpm`)
- Docker + Docker Compose
- Clerk account (free) → https://clerk.com
- Anthropic API key → https://console.anthropic.com

## Setup

### 1. Clone and install

```bash
git clone <repo>
cd secondbrain
pnpm install
```

### 2. Start the database

```bash
docker-compose up -d
```

### 3. Configure environment

Copy and fill in `apps/web/.env.local`:

```bash
cp apps/web/.env.local.example apps/web/.env.local
```

Required values:
- `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY` — from Clerk dashboard
- `CLERK_SECRET_KEY` — from Clerk dashboard
- `CLERK_WEBHOOK_SECRET` — set up webhook in Clerk → `/api/webhooks/clerk`
- `ANTHROPIC_API_KEY` — from Anthropic console
- `DATABASE_URL` — already set for local Docker

### 4. Set up Prisma

```bash
cd packages/db
npx prisma db push       # push schema to database
npx prisma generate      # generate client
```

### 5. Configure Clerk Webhook

In the Clerk dashboard:
1. Go to Webhooks → Add Endpoint
2. URL: `https://your-domain.com/api/webhooks/clerk` (or use ngrok for local dev)
3. Events: `user.created`, `user.updated`, `user.deleted`
4. Copy the signing secret → `CLERK_WEBHOOK_SECRET`

### 6. Run the app

```bash
pnpm dev
```

Open http://localhost:3000

## Adding a New Module

1. Create `modules/your-module/` directory
2. Add Prisma models to `packages/db/prisma/schema.prisma`
3. Create API routes in `apps/web/src/app/api/your-module/`
4. Create UI components in `apps/web/src/components/your-module/`
5. Add page at `apps/web/src/app/(dashboard)/your-module/page.tsx`
6. Register in sidebar `apps/web/src/components/layout/sidebar.tsx`

That's it — zero changes to core infrastructure.
